# MentorMind AI

MentorMind AI is an AI-powered placement mentor that remembers a student's learning journey and provides personalized recommendations for placement preparation.

## Features

- Student Authentication
- Learning Tracker
- AI Mentor
- Progress Dashboard
- Coding Tracker
- Personalized Study Plan

## Tech Stack

- HTML
- CSS
- JavaScript
- Python
- Flask
- MySQL
- Gemini API
